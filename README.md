# 1. Download:
```
git clone https://github.com/arturzxc/grinder.git
```

# 2. Get inside the folder
```
cd grinder
```

# 3. Compile the program:
```
g++ -g Main.cpp -lX11 -lXtst
```

# 4. Open Apex
```
Open steam and start apex and wait for it to fully start
```

# 5. Run the cheato
```
sudo ./a.out
```

# 6. Profit
```
Unlimited powaaaaaaaaaa!
```
